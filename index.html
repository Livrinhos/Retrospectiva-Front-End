Conceito Básico

HTML é a linguagem usada para estruturar o conteúdo de um site, criando elementos como títulos, parágrafos, imagens e links. Já o CSS é responsável pela aparência visual da página, permitindo definir cores, tamanhos, fontes, espaçamentos e o layout. Enquanto o HTML organiza as informações, o CSS deixa tudo mais bonito e estilizado.

Projetos com Foco em Responsividade

Projeto usando Grid

Este projeto foi criado para treinar o uso do CSS Grid. Nele, eu aprendi a organizar elementos em linhas e colunas, criando layouts mais estruturados e fáceis de controlar. O objetivo foi entender como o Grid facilita a montagem de páginas com seções alinhadas e bem distribuídas.

Projeto com Grid e Responsivo

Neste projeto, além do CSS Grid, trabalhei a responsividade usando media queries. Ajustei o layout para mudar automaticamente em telas menores, garantindo que o conteúdo continue organizado no celular. Foi um passo importante para entender como criar interfaces modernas que funcionam bem em qualquer dispositivo.

Site mais Desafiador

Esse foi o projeto que mais me deu trabalho. Tive que montar o carrinho, o checkout, o tema escuro e várias funções no JavaScript, e muita coisa deu errado até tudo funcionar. Aprendi bastante no processo e consegui entregar um site completo, mesmo sabendo que ainda poderia ter ficado melhor.
